## Overview

An AI‑powered web app to parse, summarize, and score resumes against job descriptions. It provides insights, highlights gaps, and suggests improvements.

## Stack

- Nuxt 3, Vue, Tailwind CSS
- Supabase (auth, storage)
- OpenAI/LLM provider

## Responsibilities

- Prompt design and evaluation pipeline
- UI/UX for uploading and reviewing results
- Server endpoints and rate limiting

## Highlights

- Consistent scoring with retry/pagination
- Secure file handling
- Export to PDF and JSON
