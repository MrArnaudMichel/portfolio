FL Copro

Software to manage condominiums and visit reports with speed and clarity.
Arnaud Michel
Website Git Repository

FL Copro is a focused software that streamlines how property managers prepare, conduct, and follow up on condominium visits. It centralizes reports, photos, and building records in a clear interface designed for both field work and back-office review.

Highlights

  • Efficient reports
    • Guided creation (first contact, technical visit, etc.)
    • Photos with captions grouped by section
    • Draft/Finalized with easy resume where you left off
  • Centralized condominiums
    • Building sheet with address, residents, key info
    • History of reports linked to the building
  • Productivity
    • Dashboard with filters, sorting, and “only my reports”
    • Quick search and recent files
  • Security & organization
    • Secure sign-in
    • Roles and permissions (admin, user)

Full Tech Stack

Angular 20

Standalone components, Router, and Reactive Forms for a modern SPA.

TypeScript

Strict typing for readable and maintainable code.

Node.js

Backend runtime for handling API requests and business logic.

Express

Framework for building RESTful APIs.

MongoDB

Database to store buildings, reports, and related data.

Role & Impact

  • Frontend development (Angular) and deployment automation (CI/CD)
  • Backend development with Node.js, Express, TypeScript, and MongoDB
  • Authentication and role-based permissions
  • Product and UX design (input flows, report structure, navigation)
  • Reusable design system (UI components)

Outcome: less friction during visits, faster data entry, and consistent reports that are easy to review later.

Screens & Flows

Reports dashboard

At-a-glance view with filters (type, dates, building) and sorting. A “my reports” toggle helps focus on your own work.

Create a visit report

Guided input by categories (preliminaries, structure, electricity, plumbing…). Each section supports notes, a visual state, and captioned photos.

Condominium sheet

Address, residents, key information, and access to the full history of reports associated with the building.

Launcher Studio

An open-source desktop application to create and manage application launchers (.desktop files) on GNOME-based Linux systems.
