Coco Bot

Create vertical videos clip from YouTube videos using AI.
Arnaud Michel
Full‑stack developer
Website Git Repository

Coco Bot is an AI-powered tool that transforms YouTube videos into engaging vertical clips optimized for social media platforms like TikTok and Instagram Reels. By leveraging advanced AI models, Coco Bot automates the process of content creation, making it easy to generate short, captivating videos from longer YouTube content.

Key Features

  • YouTube Integration: Seamlessly fetch videos directly from YouTube using the video URL.
  • AI-Powered Editing: Utilize AI models to analyze the video content and automatically select the most engaging segments for clipping.
  • Vertical Formatting: Convert horizontal videos into vertical format, ensuring optimal viewing on mobile devices.
  • Customizable Clip Length: Allow users to specify the desired length of the clips, catering to different social media platform requirements.
  • User-Friendly Interface: Simple and intuitive interface for users to input video URLs and receive their clips.
